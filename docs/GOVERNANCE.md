## Document Storage Requirements

All AI-generated project documents and governance artifacts must be committed into this repository under `docs/` or the relevant `ops/` subfolder. No long-lived documents may exist only outside GitHub.

---

## Phase Close-Out Checklist

Before closing any phase, verify:

- [ ] All key artifacts are present in the repository
- [ ] All documents are properly organized in appropriate folders
- [ ] All scripts and automation are tracked in Git
- [ ] All governance decisions are recorded in the Ledger
